# Analytical AI Submission – PS1

This repository contains solutions for **Problem Statement 1 (PS1)**, including both **Task 1 (Predictive Core)** and **Task 2 (Analytical Chatbot)**.

##  Task 1: Predictive Core

**Objective:**  
Analyze historical stock data and extract trends, patterns, and indicators useful for predictive modeling.

**Key Features:**
- Data ingestion and preprocessing
- Trend identification
- Analytical notebooks
- Modular utility functions

---

##  Task 2: Analytical AI Chatbot

**Objective:**  
Build a natural language chatbot that answers analytical stock-market questions using data + LLM reasoning.

**Key Capabilities:**
- Natural language query understanding
- Ticker detection (AAPL, MSFT, TSLA)
- Trend-based explanations
- Retrieval-Augmented Generation (RAG)
- Vector database using Chroma
- LLM integration via Groq

**Example Queries:**
- *Why did Apple stock fall?*
- *When did Microsoft stock rise?*
- *Explain Tesla’s recent trend*

---

## ⚙️ Tech Stack

- Python
- LangChain
- Groq LLM
- HuggingFace Embeddings
- Chroma Vector DB
- Pandas / NumPy
